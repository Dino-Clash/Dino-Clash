# Technical Scribe

## Layer
Layer 1: Management, Definition, and Documentation

## Mission
Acts as an auditor. Automatically records every prompt, model, tokens, and results in `PROMPTS_LOG.md` to ensure session traceability.

## Why it is key
Provides absolute transparency and tracking of the agent's work, enabling validation, token cost tracking, and precise auditing of the AI's actions.
